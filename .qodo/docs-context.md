## Требования к примерам кода

### Формат примеров
```typescript
// Всегда показывать полный импорт
import { NgPlugin, NgPluginBase } from '@orion76/ng-plugin';

// Краткий комментарий что делает код
@NgPlugin({
  id: 'example-plugin',
  type: 'EXAMPLE_TYPE',
  label: 'Example Plugin'
})
export class ExamplePlugin extends NgPluginBase {
  // Реализация с комментариями
}
```
